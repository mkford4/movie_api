Movie API

This application provides users with access to information regarding different movies, including various details such as directors, year of release, and top-billed actors. Similar to IMDb.

Main Features

Request URL HTTP Method Response
Return a list of ALL movies to the user '/movies' GET Returns a JSON list of all movies on file
Return data about a single movie by title to the user '/movies/:title' GET Returns information about a single movie via JSON
Return data about a genre by movie title '/movies/genre/:genreName' GET Returns information/description of specified genre via JSON
Return data about a director (bio, birth year, death year) by name '/movies/directors/:directorName' GET Returns information regarding specified director with JSON
Allow new users to register '/users/:newUser' POST Adds new user to JSON object
Allow users to update their user info (username) '/users/:username' PUT Returns updated username with text message confirmation
Allow users to add a movie to their list of favorites '/users/:username/favorites' PUT Returns text message with confirmation movie added to list
Allow users to remove a movie from their list of favorites '/users/:username/favorites' DELETE Returns a text message confirming that movie has been removed
Allow existing users to deregister '/users/:username' DELETE Returns a text message that user email has been removed
