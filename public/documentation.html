Movie API

This application provides users with access to information regarding different movies, including various details such as directors, year of release, and top-billed actors. Similar to IMDb.

Main Features

Request URL HTTP Method Response
Return a list of ALL movies to the user '/movies' GET Returns a JSON list of all movies on file
Return data about a single movie by title to the user '/movies/:Title' GET Returns information about a single movie via JSON
Return data about a genre by movie title '/movies/Genre/:Name' GET Returns information/description of specified genre via JSON
Return data about a director (bio, birth year, death year) by name '/movies/Directors/:Name' GET Returns information regarding specified director with JSON
Allow new users to register '/users' POST Adds new user to JSON object
Return a list of ALL movies '/users' GET Returns a JSON list of ALL users on file
Get all data of a user by Username '/users/:Username' GET Returns JSON data on a specified user
Allow users to update their user info by username via JSON object '/users/:Username' PUT Returns updated username with text message confirmation
Allow users to add a movie to their list of favorites '/users/:Username/movies/:MovieID' POST Returns text message with confirmation movie added to list
Allow users to remove a movie from their list of favorites '/users/:Username/movies/:MovieID' DELETE Returns a text message confirming that movie has been removed
Allow existing users to deregister '/users/:Username' DELETE Returns a text message that user email has been removed
